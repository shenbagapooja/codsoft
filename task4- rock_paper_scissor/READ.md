# Rock-Paper-Scissors Game 🎮

A simple, interactive command-line implementation of the classic **Rock-Paper-Scissors** game written in Python. It includes score tracking, input validation, and a fun, personalized user interface with emojis!

______________
##🎯About the Project

This Python script is an interactive, console-based implementation of the classic **Rock-Paper-Scissors** game. It allows a user to play multiple rounds against an automated computer opponent, tracks scores dynamically, and features a customized, friendly terminal interface.

_____________

## 📌 Main Uses

* **Programming Practice:** It serves as an excellent foundational project for learning core Python concepts like `while` loops, conditional `if-elif-else` logic, and user input validation.
* **Console Entertainment:** It acts as a quick, lightweight command-line game that can be run directly inside any terminal or IDE.

_______________
## 🎮 How to Play
1. Run the script in your terminal.
2. Type `rock`, `paper`, or `scissors` when prompted.
3. The computer will randomly make its choice, and the winner of the round will be displayed.
4. To exit the game at any time, simply type `quit`.
________________

##  📌Advantages

* **Robust Input Handling:** The code uses `.lower().strip()` and validation checks to prevent invalid inputs from crashing the program.
* **Dynamic Score Tracking:** It stores scores across multiple rounds so players can see their overall performance instead of just a single-round result.
____________________
## 🏷️Features
* **Interactive Gameplay:** Play against the computer in a terminal environment.
* **Score Tracking:** Keep track of your wins and the computer's wins across multiple rounds.
* **Input Validation:** Handles typos or incorrect inputs gracefully without crashing.
* **Replay Option:** Prompted at the end of each round to continue playing or exit.

__________________
