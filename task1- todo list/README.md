# 💻 To-Do List App

A clean, lightweight desktop To-Do List application built with Python and Tkinter — with absolutely no external dependencies required.
___________
## 🎯About

- A lightweight desktop task manager built with **Python** and **Tkinter**
- Requires zero external dependencies — runs on pure Python
- Designed to be simple, clean, and beginner-friendly
- Great starting point for learning Python GUI development
- Covers key concepts like event binding, listbox manipulation, and dialog popups
- Works on **Windows**, **macOS**, and **Linux**

---
## How to Use
1. Run the app using `python todo.py`
2. Click the input field — placeholder text will disappear automatically
3. Type your task and click **Add Task** to add it to the list
4. To mark a task complete, select it from the list and click **Mark as Done** — a ✔️ will appear
5. To remove a task, select it and click **Delete Task**
6. If you perform an action without selecting a task, a warning popup will guide you
---
## ✨ Features

* **➕ Add Tasks:** Quickly add new items via a text input field with placeholder support.
* **✔️ Complete Tasks:** Mark tasks as done with a clear checkmark indicator.
* **🗑️ Delete Tasks:** Easily remove individual tasks from your list.
* **💬 Smart Alerts:** Friendly warning dialogs prevent invalid actions.
* **🚫 Pure Python:** Built entirely using the standard library (`tkinter`).
