## Introduction
Welcome to this repository containing a collection of four distinct, beginner-friendly Python applications. These projects cover a range of functionalities, from command-line games and utility scripts to a Graphical User Interface (GUI) desktop application.


## Projects Overview

### 1. To-Do List
Built this one using Python's `tkinter` library, so no need to install anything extra. It's a straightforward desktop app for managing daily tasks without any hassle.

* **Key Features:**
    * Simple graphical interface that's easy to use
    * Placeholder text in the input box so you know what to type
    * You can add tasks, delete them, or mark them as done
    * Shows error popups instead of crashing when something goes wrong
* **File:** `todo list.py`

### 2. Simple Calculator
Nothing fancy — just a clean console calculator that handles everyday math. Sometimes simple is all you need.

* **Key Features:**
    * Asks for two numbers and an operator to work with
    * Handles addition, subtraction, multiplication, and division
    * Won't crash if you accidentally try to divide by zero
* **File:** `calculator.py`

### 3. Password Generator
A handy little script for when you need a strong password quickly. Just tell it how long you want the password and it does the rest.

* **Key Features:**
    * Mixes lowercase, uppercase, numbers, and special characters
    * Length is fully customizable based on what you need
    * Output is random enough that it's hard to predict or guess
* **File:** `password generator.py`

### 4. Rock-Paper-Scissors Game
A fun little command-line game where you go head-to-head against the computer in as many rounds as you want. I made sure it covers the basics without overcomplicating things.

* **Key Features:**
    * Validates your input so the game doesn't break if you type something weird
    * Computer picks its move randomly, so it actually feels fair
    * Keeps score across multiple rounds
    * Lets you choose to play again or quit at any point
* **File:** `rock-paper-scissor.py`
