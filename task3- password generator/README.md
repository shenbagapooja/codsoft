# Password Generator-task 3

## 🎯About
This is a simple python program I made for Task 3. What it does is ask the user to enter a number and then it generates a random password using that number as the length. It uses letters, numbers and symbols to make the password.

## 📌Uses
* Can be used to create strong passwords
* Helpful when you don't want to think of a password yourself
* Good for any website or app that needs a password

##📌 How to Use
1. Run the program in python
2. It will ask you to enter the length of the password
3. Type a number and press enter
4. The password will be shown on the screen
